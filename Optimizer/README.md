===================================================
HYPERION MULTI-INSTANCE OPTIMIZER
===================================================
HOW IT WORKS:
- The FIRST Roblox window you open is automatically protected as your "MAIN" account. It will never be optimized or touched.
- Every Roblox window opened AFTER the first one is marked as an "ALT".
- After 30 seconds of loading, all ALTs are forced to minimize, assigned to specific background CPU cores, and their RAM is aggressively trimmed down.

HOW TO USE:
1. Open your MAIN Roblox account first. Wait for it to load.
2. Open all your ALT accounts.
3. Double-click the "RunOptimizer.bat" file.
4. Click "Yes" to the Administrator prompt (it needs this to manage memory/CPU).

CONFIGURING:
A file called "HyperionOptimizer.config.json" will be created the first time you run it. You can edit this file to change the memory limits, how long it waits before optimizing, and how many CPU cores the bots are allowed to use.

REQUIREMENTS:
- Windows 10 or 11
- Administrator privileges
